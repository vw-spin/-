VW-Spin

Este é um projeto da turma 2AP-B do Centro de Formação SENAI Volkswagen para a disciplina REDIM. O site é uma **loja interativa de cubos mágicos**, onde os clientes podem ver os produtos, fazer pedidos e acompanhar o rastreamento de cada compra.

## Tecnologias Utilizadas

Para construir este site, foram usadas as seguintes tecnologias:

- **HTML** para a estrutura das páginas  
- **CSS** para estilos visuais e responsividade  
- **JavaScript** para a interatividade (carrinho, rastreamento, formulários)  
- **Firebase** banco de dados para armazenar pedidos
